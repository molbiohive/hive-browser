# Zerg browser

Zerg browser is a simple and fast app designed to quickly scan directories for biological data, such as DNA, RNA and protein sequences, which thne can be searcher with web interface with fuzzy search.

It aim to simplify and improve search for desired sequence of feature, without manually looking for it.

## Why Zerg?

Zerg is a reference to type of a create from StarCraft universe, which are known for their fast multiplication, mutations and adaptations. This browser leverages the concept to be a tool to control this beast (i.e. rapidly changing data) with it's traits.
